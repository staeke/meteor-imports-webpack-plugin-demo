Meteor app running with webpack!

❤️

Great! This page is to demo a setup where:

  • The client is built using webpack
  • The server runs as a normal meteor app (but not the client parts)
  • You can run the full app as a normal Meteor app inside meteor-app. There are some risks involved with this, so see the README.md file
  • images and media are served using fallback routes in webpack-dev-server to meteor-app/public. To build them using webpack, use the HtmlPlugin
  • Twitter bootstrap is used as a Meteor package for demo purposes. NOTE: These types of packages are really recommended to include outside of Meteor packages
  • Various scripts can be found in package.json
